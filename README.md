# MyLove

A psychological roguelike set inside the mind of a protagonist trapped in a coma.

## About

MyLove explores themes of fear, loneliness and self-acceptance.

The player is accompanied by five distinct personalities, each representing a different aspect of the protagonist's psyche.

Every personality grants unique gameplay abilities and influences the story.

## Personalities

- Critic
- Child
- Rebel
- Savior
- Anxiety

## Gameplay

- Roguelike progression
- Character development
- Multiple endings
- Personality-based abilities
- Story-driven exploration

## Current Status

🛠 Concept and Prototype Stage (Stopped)

## Goals

Create a game where gameplay systems and narrative work together to reflect the protagonist's internal struggles.
